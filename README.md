# Mind-Breaker: The Ultimate 120-Question Challenge

**Mind-Breaker** is a web-based, mobile-friendly, interactive quiz game inspired by the classic “Impossible Quiz.”  
It features **120 questions in a fixed, set order**, including medium/easy questions, trick questions, playable mini-games, and super hard final challenges.  

This project is **open-source**, meaning anyone can freely use, modify, and share it under the terms described below.

---

## Features

- **Fixed Question Order**
  - Every player experiences the same sequence of questions
  - Difficulty progresses from easy to medium, with trick questions and playable challenges strategically placed
  - Last 10 questions are super hard, at the end of the sequence

- **120 Questions Total**
  - 100 medium/easy slightly tricky questions  
  - 27 trick questions at fixed positions  
  - 10 interactive playable questions (mini-games or normal tricky questions)  
  - 10 super hard questions (questions 111–120)

- **Mini-Games**
  - Tap-and-hold challenges  
  - Drag-and-drop tasks  
  - Fun interactive moments embedded in playable questions

- **Lives System**
  - Players start with 3 lives (❤️❤️❤️)  
  - Wrong answers or failed mini-games subtract a life  
  - Game ends if all lives are lost

- **Mobile-Friendly**
  - Large buttons for easy tapping  
  - Responsive layout for all screen sizes  
  - Touch and swipe support for mini-games

- **Fail Animations**
  - Wrong answers trigger visual feedback  
  - Keeps gameplay engaging and fun
